Hotel Bookings Project:

This project was created in my second year of university for my Web Development final assignment. The website is connected to a database that would upload the data submitted from a form displayed on the website, it would then save the information onto the database and display the details submitted onto a separate page on the website. This website was created using HTML, CSS and PHP. (MY)SQL was used to gather/retrieve information to/from the database.
